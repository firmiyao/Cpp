# 栈：

**栈（stack）是一种只能在一端进行插入的或者删除操作的线性表**

**允许进行插入，删除操作的一端叫做栈顶，表的另一端叫做栈底。**

**栈的主要特点是“后进先出”，所以栈也叫“后进先出表”。**

## 栈的声明与实现：

栈的功能应该如下：

+ 初始化栈
+ 销毁栈
+ 判断栈是否为空
+ 进栈
+ 出栈
+ 取得栈顶元素的值

头文件如下：

*define.h*:

```c++
#ifndef exa_h
#define exa_h

struct exa {
	int data[50];
	int top;
};

void exa_init(exa*& L);  //初始化
void exa_destory(exa* L);   //销毁栈
bool exa_isempty(exa* L);   //判断栈是否为空
bool exa_push(exa* L, int n);    //进栈
bool exa_pop(exa* L, int& n);    //出栈
bool exa_top(exa* L, int& n);   //取栈顶元素
#endif
```

栈的四个性质：

+ **栈空的条件：s->top==-1;**
+ **栈满的条件：s->top==Maxsize-1**
+ **元素e的进栈操作：先将栈顶指针top增1，然后将元素e放在栈顶指针处**
+ **出栈操作：先将栈顶指针top处的元素取出放在e中然后将栈顶元素减1**

## 定义：

```c++
#include<iostream>
using namespace std;
#include"define.h"

void exa_init(exa*& L)
{
	L = new exa;
	L->top = -1;
}

void exa_destory(exa* L)
{
	delete L;
}

bool exa_isempty(exa* L)
{
	return L->top == -1;
}

bool exa_push(exa* L, int n)
{
	if (L->top == 50 - 1)
		return false;
	L->top++;
	L->data[L->top] = n;
	return true;
}

bool exa_pop(exa* L, int& n)
{
	if (L->top == -1)
		return false;
	n = L->data[L->top];
	L->top--;
	return true;
}

bool exa_top(exa* L, int& n)
{
	if (L->top == -1)
		return false;
	n = L->data[L->top];
	return true;
}
```

例题：**判断一个字符串是否是对此字符串**

```c++
#include<iostream>
using namespace std;
#include"define.h"
int main(void)
{
	exa* L;
	exa_init(L);
	char s[30];
	cin >> s;
	for (int i = 0; s[i] != '\0'; i++)
		exa_push(L, s[i]);
	for (int i = 0; s[i] != '\0'; i++)
	{
		char t;
		exa_pop(L, t);
		if (t != s[i])
		{
			exa_destory(L);
			cout << "false" << endl;
			return 0;
		}
	}
	cout << "true" << endl;
	return 0;
}
```

# 链栈：

**采用链式存储结构的栈称为链栈**

**链栈的优点是不存在栈满溢出的情况**

## 链栈的声明与实现：

链栈的性质：

s为头节点

+ **栈空的条件：s->next==NULL**
+ **栈满的条件：通常不考虑**
+ **元素e的进栈操作：新建一个结点存放元素e，并插入到头节点后**
+ **出栈操作：取出首结点的data值并将其删除**

## 链栈的声明与定义：

*define.h*:

```c++
#ifndef exa_h
#define exa_h
typedef int Type;
struct exa {
	int data;
	exa* next = NULL;
};

void exa_init(exa*& L);  //初始化
void exa_destory(exa* L);   //销毁栈
bool exa_isempty(exa* L);   //判断栈是否为空
bool exa_push(exa* L, int n);    //进栈
bool exa_pop(exa* L, int& n);    //出栈
bool exa_top(exa* L, int& n);   //取栈顶元素
#endif
```

## 定义：

```c++
#include<iostream>
using namespace std;
#include"define.h"

void exa_init(exa*& L)
{
	L = new exa;
}

void exa_destory(exa* L)
{
	exa* t = L;
	while (t->next != NULL)
	{
		exa* u = t->next;
		delete t;
		t = u;
	}
	delete t;
}

bool exa_isempty(exa* L)
{
	return L->next == NULL;
}

bool exa_push(exa* L, int n)
{
	exa* t = new exa;
	t->data = n;
	t->next = L->next;
	L->next = t;
	return true;
}

bool exa_pop(exa* L, int& n)
{
	if (L->next == NULL)
		return false;
	n = L->next->data;
	exa* t = L->next;
	L->next = t->next;
	delete t;
	return true;
}

bool exa_top(exa* L, int& n)
{
	if (L->next == NULL)
		return false;
	n = L->next->data;
	return true;
}
```
