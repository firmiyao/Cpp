# C++内联函数：

+ **内联函数是C++为了提高程序运行速度所做的一项改进。常规函数和内联函数之间的主要区别不在于编写方式，而在于C++编译器如何将他们组合到程序中**

+ **对于常规函数的调用，根据代码的长度需要进行一系列的调用，而内联函数无需跳到另一个位置执行代码，再跳回来，所以速度更快，不过代价是要付出更多的内存**
+ **记住：如果执行函数代码的时间的处理函数调用机制的时间长，则节省的时间将只占整个过程的很小部分。如果代码执行的时间很短，则内联调用就可以节省非内联调用使用的大部分时间**
+ 内联函数不能递归

关键字：***inline***

措施：

+ 在函数声明前加上inline
+ 在函数定义前加上inline

展示：

```c++
#include<iostream>
using namespace std;
inline int fun(int n);    //函数声明前加inline
int main()
{
    cout << fun(3);
    return 0;
} 
inline int fun(int n)    //函数定义前加inline
{
    return n * n;
}
```

结果为9；**如果内联函数过大或者调用了自己，那么编译器不会满足这种要求。**

# 引用变量：

C++新增了一种复合类型——引用变量。**引用是已定义的变量的别名**

**引用变量的主要用途是用作函数的形参。通过将引用变量用作参数，函数将使用原始数据，而不是其副本。（与指针类似）**

## 创建引用变量：

关键字：***&***，这里不指取地址符。

上才艺：

```c++
#include<iostream>
using namespace std;
int main()
{
    int n = 8;
    int& m = n;   //n的别名为m；
    cout << "m = " << m << endl;
    cout << "&n = " << &n << endl;
    cout << "&m = " << &m << endl;    //这里的&是取地址符
    n++;
    cout << "n = " << n << endl;
    cout << "m = " << m << endl;
    m++;
    cout << "n = " << n << endl;
    cout << "m = " << m << endl;
    return 0;
} 
```

结果如上，可知：

+ m和n的地址和值都相等
+ 两个值只要一个变，另一个也变化

指针和引用有一定的相似性，但也有差别，比如：**必须在声明引用时将其初始化，而不能像指针那样，先声明，再赋值。引用更接近于 const 指针，必须在创建时初始化，一旦与某个变量关联起来，就将一直效忠于他。

```c++
#include<iostream>
using namespace std;
int main()
{
    int n = 8;
    int& m;   
    m = n;
    return 0;
} 
```

这是错误的做法，不能先声明再赋值。

```c++
int & m = n;
等效于：
int * const m = & n ;
```

理解以下程序：

```c++
#include<iostream>
using namespace std;
int main()
{
    int n = 8;
    int& m = n;
    cout << "n = " << n << endl;
    cout << "m = " << m << ", &m = " << &m << endl;
    int c = 3;
    m = c;
    cout << "c = " << c << endl;
    cout << "m = " << m << ", &m = " << &m << endl;
    c++;
    cout << "c = " << c << endl;
    cout << "m = " << m << ", &m = " << &m << endl;
    return 0;
} 
```

## 将引用用作函数参数：🔴

C++新增的这项特性是对C语言的超越，C语言只能按值传递。按值传递导致被调用函数使用调用程序的值的拷贝。

分析**非引用参数，引用参数和指针参数**上才艺：

```c++
#include<iostream>
using namespace std;
void fun_1(int a)   //非引用
{
    a = 4;
}
void fun_2(int &n)   //引用
{
    n = 5;
}
void fun_3(int* p)   //指针
{
    *p = 8;
}
int main()
{
    int a = 1;
    fun_1(a);
    cout << a<< endl;
    fun_2(a);
    cout << a<< endl;
    fun_3(&a);
    cout << a << endl;
}
```

# 将引用用于结构：

**使用结构引用参数的方式于使用基本变量引用相同，只需在声明结构参数时使用引用运算符&即可。**

```c++
#include<iostream>
using namespace std;
struct exa {
    int n;
};
int main()
{
    exa i = { 1 };
    cout << "i.n = " << i.n << endl;
    exa& j = i;  //结构的引用
    cout << "j.n = " << j.n << endl;
}
//i.n和j.n 均为1
```

## 结构引用用于函数

如果不希望函数修改传入的结构，可以使用const。

下面的程序返回**指向结构的引用**，上才艺：

```c++
#include<iostream>
#include<cstring>
using namespace std;
struct exa {
    char name[20];
};
void init(exa& g)
{
    cout << "Enter the name:" << endl;
    cin.getline(g.name, 20);
}

//这里不改变结构的值，只是展示，所以可以加const防止修改
void show(const exa g)   
{
    cout << "The name is :" << g.name << endl;
}
exa& copy(exa& v, const exa c)  //返回一个指向结构的指针
{
    strcpy(v.name, c.name);
    return v;     //感觉只是返回一个结构，但这里返回的是一个指向结构的引用
}
int main()
{
    exa i;
    init(i);
    show(i);
    exa r;
    cout << "Begin copy:" << endl;
    copy(r, i);
    show(r);
    return 0;
}
```

### 为什么要返回指向结构的引用？

**因为更节省空间**

就好比：

```c++
int n=2*2;    将结果拷贝给n
cout<<n;
和
cout<<2*2;    没有拷贝结果
```

放回引用节省了将结果拷贝给另一个值得空间

注意：**返回引用时，应该避免返回函数终止时不再存在得内存单元应用。**应该避免写下面这样的代码：

```c++
exa & fun(exa & j)
{
    exa i;
    .....
    return i;
}
```

**如上是返回了一个指向结构的指针，不过函数运行后结构i的空间将不复存在，引用失效。同时也要避免返回指向临时地址的指针。**可以使用new。

```c++
#include<iostream>
#include<cstring>
using namespace std;
struct exa {
    char name[20];
};
//这里不改变结构的值，只是展示，所以可以加const防止修改
void show(const exa g)   
{
    cout << "The name is :" << g.name << endl;
}
exa& copy(const exa c)  //返回一个指向结构的指针
{
    exa* v = new exa;
    strcpy((*v).name, c.name);
    return (*v);     //感觉只是返回一个结构，但这里返回的是一个指向结构的引用
}
int main()
{
    exa c = { "wtf" };
    show(c);
    exa v;
    v = copy(c);
    show(v);
    return 0;
}
```

该程序可以正确运行，不过有缺陷，就是没有用delete删除new申请的空间。

# 默认参数：

**默认参数是指当函数调用中省略了实参时自动使用的一个值**

假设有一个函数fun，它有三个整形形参a,b,c则：

```c++
void fun(int a,int b,int c)   //常规做法
void fun(int a,int b=0,int c=3)   //合法
void fun(int a,int b=0,int c)  //不合法
void fun(int a=1,int b=3,int c=4)   //合法
```

所以，**要为某个参数设置默认值，则必须为它右边的所有参数提供默认值**

编写一个展现字符串前n个字符的程序，默认n为5，上才艺:

```c++
#include<iostream>
#include<cstring>
using namespace std;
char* fun(const char* t, int n = 5) //返回一个指针
{
    char* p = new char[n + 1];
    char* h = p;
    while (n--)
    {
        *p = *t;
        p++;
        t++;
    }
    *p = '\0';
    p = h;
    return p;
}
int main()
{
    char str[40];
    cin.getline(str, 40);
    char* p;
    p = fun(str); //使用默认值
    cout << p << endl;
    delete[]p;
    char* n;
    n = fun(str, 7);
    cout << n << endl;
    delete[]n;
    return 0;
}
```

# 函数重载：

**函数多态是C++在C语言的基础上新增的功能。**

**函数多态（函数重载）让我们可以使用多个同名的函数。术语“多态”指的是有多种形式，因此函数多态允许函数可以有多种形式。类似的，函数重载指的是可以有多个同名的函数，因此对名称进行了重载。这都是一回事，记住，它们完成相同的工作，但是使用不同的参数列表。**

函数重载的关键是函数的**参数列表**——也称为**函数特征标**。如果两个函数的**参数数目**和**类型**相同，同时**参数的排列顺序**也相同，则它们的特征标相同，而变量名无关紧要，毕竟都是同名？

上才艺：

```c++
#include<iostream>
#include<cstring>
using namespace std;
//定义四个同名函数：
void fun(int a)
{
    cout << "The 1 ." << endl;
}
int fun(char c)
{
    cout << "The 2 ." << endl;
    return 0;
}
void fun(int n, char c)
{
    cout << "The 3 ." << endl;
}
char fun(char c, int n)
{
    cout << "The 4 ." << endl;
    return '\0';
}
int main()
{
    fun('b', 5);
    fun(8, 'l');
    fun('f');
    fun(7);
    return 0;
}
```

![image-20221011172521391](https://cdn.jsdelivr.net/gh/firmiyao/Picture/img/202210111725421.png)

**如上四个函数，他们同名，但是特征标不同，而且返回类型也不同，但是返回类型并不是关键。**



**注意：匹配函数时，要注意const和非const 。注意将非const值赋给const参数是合法的，但是反过来却不行**

```c++
#include<iostream>
#include<cstring>
using namespace std;
//定义四个同名函数：
void fun(int *p)
{
    cout << 1;
}
int main()
{
    int i=2;
    const int* p = &i;
    fun(p);
    return 0;
}
```

如上将会报错：

![image-20221011174509311](https://cdn.jsdelivr.net/gh/firmiyao/Picture/img/202210111745339.png)

若有如下代码：

```c++
#include<iostream>
#include<cstring>
using namespace std;
//定义四个同名函数：
void fun(int *p)
{
    cout << 1<<endl;
}
void fun(const int* p)
{
    cout << 2 << endl;
}
int main()
{
    int a = 1;
    int* p = &a;
    int b = 2;
    const int* q = &b;
    fun(p);    //输出1
    fun(q);    //输出2
    return 0;
}
```

可见同名函数fun，它的形参分别为const和非const，这是编译器会选择最优的方法，若实参为const就用第二个fun，反之亦然。

结果如下为1，2.
